# Sigma Engine

Sigma Engine is a Psych Engine 1.0.4-compatible Friday Night Funkin' engine fork. It keeps the familiar Psych Engine source layout, mod folder workflow, Lua and HScript support, Mods menu, Weekend content, and editor tools while building as `SigmaEngine.exe`.

## Build

Read [`README_SIGMA_ENGINE.txt`](README_SIGMA_ENGINE.txt) first. On Windows, run `setup\\windows.bat` once and then run `BUILD_SIGMA_ENGINE_X64.bat`.
